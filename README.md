# hubot-dvb

hubot script to get current dvb information

See [`src/dvb.coffee`](src/dvb.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-dvb --save`

Then add **hubot-dvb** to your `external-scripts.json`:

```json
[
  "hubot-dvb"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
